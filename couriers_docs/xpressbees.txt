POST Ndr List
https://shipment.xpressbees.com/api/ndr/create
Use this API endpoint to generate Take action against any AWB. Action against 100 AWBs are allowed in one request.
Note: Action is allowed only when Exception is raised by courier company.
Supported Action Types: re-attempt, change_address, change_phone
Check sample request body for supported action_data for each action type.
